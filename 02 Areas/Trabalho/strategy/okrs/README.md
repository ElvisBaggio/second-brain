# OKRs — Histórico

Cada ciclo tem seu próprio arquivo. Convenção de nome: `YYYY-QN.md` (trimestral) ou `YYYY-HN.md` (semestral).

## Convenção de arquivo OKR

```markdown
# OKR [Período]

## Objetivo 1: [Título orientado a impacto]
- KR1: [Métrica] de X para Y até [data]
- KR2: ...

**Status:** 🟢 on track | 🟡 risco | 🔴 off track
**Última atualização:** YYYY-MM-DD
```

## Formato de referência

Veja `[[OKR Q2-2026 - Tribo A]]` no Produto MGC como exemplo de OKR por pilar bem estruturado (objetivo + KRs com baseline e target + progresso semanal).
