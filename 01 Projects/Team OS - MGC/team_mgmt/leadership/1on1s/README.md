# 1:1s com Lideranças

Mesmo formato de `team_mgmt/1on1s/`, mas focado em líderes com escopo de gestão.

A diferença de pauta em relação a 1:1s com ICs:

| IC (individual contributor) | Liderança |
|---|---|
| Projetos e entregas | Saúde do time que lidera |
| Desenvolvimento pessoal | Bloqueios sistêmicos |
| Feedback direto | Alinhamento de visão |
| Carreira | Decisões de pessoas |

## Convenção de arquivo
`1on1-[nome].md` — mesmo padrão de `team_mgmt/1on1s/`

## Template de entrada para líderes

```markdown
### [YYYY-MM-DD]
**Humor/energia:** 🟢 alto | 🟡 médio | 🔴 baixo

**Saúde do time:**
- ...

**Bloqueios que preciso destravar:**
- ...

**Alinhamento de visão / prioridade:**
- ...

**Ações combinadas:**
- [ ] (responsável) ...
```
