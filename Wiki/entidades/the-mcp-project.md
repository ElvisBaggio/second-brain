# The MCP Project

Organização/iniciativa responsável pela criação e manutenção do [[Model Context Protocol]] (MCP).

## O que mantém

- A especificação aberta do protocolo MCP
- O blog oficial em [blog.modelcontextprotocol.io](https://blog.modelcontextprotocol.io/)
- O processo de [[Specification Enhancement Proposal|SEPs]] para evolução da spec

## Contexto

O MCP foi criado pela Anthropic e posteriormente aberto à comunidade como protocolo independente. O "MCP Project" representa a iniciativa de governança aberta do protocolo.
